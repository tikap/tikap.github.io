About

About the author

I am Tika Pradhan, a computer engineering graduate from the University of Guelph. I have completed IT contract positions after university and, proceeded to work full time as a software developer.

What is this website?

This is an experimental website created by me from scratch without using Wix, WordPress or other third party site-building services. I built it with the goal to learn the fundamentals of front-end development, version control and how everything interacts with each other.

What tools did you use to build this site?

I coded the backbone of this site using the classic trio of HTML, CSS and JavaScript. I chose BootStrap to make it responsive across all platforms (try visiting the site on your phone, satisfaction guaranteed!)

What's the future of this site?

When you read this, the content in the rest of the pages are probably mostly empty. As mentioned above, the main intention of this site is for me to learn.

Additionally, you can probably see from the URL that I am using a free and limited web domain server provided by GitHub pages (https://pages.github.com/). I look forward to exploring deeper into back-end development with database and server communications and publish this site to the internet away from GitHub pages and on my own domain (Tika!).
