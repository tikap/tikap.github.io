About

Who made this site?

I built and authored this site. It's disguised as a blog riddled with tech content but in actuality, it is a playground for me to experiment site design, coding and exploring various technologies in the computing world.

More about the Author.

My name is Tika Pradhan, a graduate from the University of Guelph. I studied Computer Engineering and worked in many contract positions, usually in the Information Technology fields. I have yet to work in a full time job, so I look forward to that step in my professional career!

What is this website?

This is an experimental website created by me from scratch without using Wix, WordPress or other third party site-building services. I built it with the goal to learn the fundamentals of front-end development, version control and how everything interacts with each other.

What tools did you use to build this site?

I coded the backbone of this site using the classic trio combo of HTML, CSS and JavaScript. I chose BootStrap to style this site responsive and well designed across all platforms (try visiting the site on your phone, satisfaction guaranteed!)

What's the future of this site?

When you read this, the content in the rest of the pages are probably mostly empty. I plan to write technological posts as well as work on optimising this site at the same time. As mentioned above, the main intention of this site is for me to learn, so there is no deadlines as to when I will finish the "meat" of this site.

My goal is to polish this site's design and add a bit of content in all three categories. At this time, you can probably see from the URL that I am using a free and limited web domain server provided by GitHub pages (https://pages.github.com/). I look forward to exploring deeper into back-end development with database and server communications and publish this site to the internet away from GitHub pages and on my own domain (Tikasity!).
